一些提醒

以下提醒根据22学年下学期的规则（包括一些内部规则）。

如果规则更新了记得踢我更新，高三了不太管这些。

  • 打卡不是刷脸的，天线是脚下那两条长的，不要挤在机子面前刷了。
  • 目前代跑是按刷卡时间算的，班主任会收到“疑似代跑”名单。反正暂时没有硬性地处理代跑就好了。
  • 单次配速要求 九分钟 之内否则不计入，前几圈跑慢了可以紧接着的下一圈跑快点。
  • 单次路程要求大于 1km 否则不计入，早上打卡那 0.25km 是不算数的。
  • 路程按打了几个点算，来回走什么的都是可以的。
  • 查自己成绩可以在 https://jinhuaschool.smart-run.cn/report 查，高一的号码推算不了了，高二和高三可以推。
  • 实在不知道自己学号可以问我要，没 Telegram 就在 GitHub 上面提 Issue 好了。我会回的。
  • 或者你可以让自己上榜就知道自己跑了多少了。
  • 我没选技术不要问我技术学科相关问题。我就是个臭写代码的。
